* SQLite
* HTML
* CSS

### Features

* User Registration
* User Login
* Role-Based Authentication
* Vendor Dashboard
* Add Product
* View Products
* Delete Product
* Customer Dashboard
* Product Browsing

### Project Description

A multi-vendor e-commerce marketplace built using Python Flask and SQLite. Vendors can register, log in, add products, view products, and delete products. Customers can register, log in, browse products, and access a customer dashboard.
